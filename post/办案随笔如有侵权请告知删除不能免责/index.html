办案随笔|如何简单的验证对方身份证号码真伪？

作者: 黄为春 写于 2023-05-06

双方签订合同时，一般需要填写双方的名字与身份证号码，以避免引起纠纷后，起诉时找不到当事人。

如果双方在签订合同时，没有提供身份证复印件，只填写了身份证号码，如何验证身份证号码的真伪？

我们可以根据身份证号码构成来初步判断，依据我国公民身份证号码的国家标准可知1，身份证号码的构成为：

地址码（6位） + 出生日期码（8位） + 顺序码（3位） + 校验码（1位）

  • 地址码：表示常住户口所在县（市、旗、区）的行政区划代码。
  • 出生日期码：表示编码对象出生的年、月、日，按GB/T 7408 的规定执行。年、月、日代码之间不用分隔符。
  • 顺序码：表示在同一地址码所标识的区域范围内，对同年、同月、同日出生的人编定的顺序号，顺序码的奇数分配给男性，偶数分配给女性。
  • 校验码：校验码根据一定的公式计算而得，校验码采用 ISO 7064:1983,MOD 11-2 校验码系统。

所以，我们可以通过地址码、出生日期码、顺序码的奇偶数大致判断身份证号码是否真实。

其中地址码是非常有规律的:

  • 第1位数字表示地区，1表示华北，2表示东北，3表示华东，4表示中南，5表示西南，6表示西北，7表示台湾地区，8表示香港澳门地区。

  • 第2位数字表示户籍地所在的直辖市、省、自治区在该地区中的排位，比如：
    华北地区：北京市-11，天津市-12，河北省-13，山西省-14，内蒙古自治区-15
    东北地区： 辽宁省-21，吉林省-22，黑龙江省-23
    华东地区： 上海市-31，江苏省-32，浙江省-33，安徽省-34，福建省-35，江西省-36，山东省-37
    华中地区： 河南省-41，湖北省-42，湖南省-43
    华南地区：广东省-44，广西壮族自治区-45，海南省-46
    西南地区：重庆市-50，四川省-51，贵州省-52，云南省-53，西藏自治区-54
    西北地区： 陕西省-61，甘肃省-62，青海省-63，宁夏回族自治区-64，新疆维吾尔自治区-65

  • 第3-6位则表示到具体的县（市、旗、区），具体的代码可以在中国民政部官网查询2。

当然，这种方式只能初步验证，并不能真正核实真伪，如果与他人签订合同，必须用身份证原件进行核对，并留存身份证复印件以备不时之需。
